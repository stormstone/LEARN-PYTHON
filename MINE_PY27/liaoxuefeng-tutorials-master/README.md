# liaoxuefeng-tutorials

廖雪峰制作的 Git, Python 2/3, JavaScript 精品教程，可读性很强。

可惜官方只提供了在线版本，不方便离线阅读，于是写了这个脚本来抓取。

## 功能简介

从廖雪峰的官方网站获取 Git, Python 2/3, JavaScript 教程，保存为 HTML 格式，可用于制作离线电子书

保存格式：单个 HTML 文件，并下载所有图片附件

## 脚本说明

使用的 Python 版本：

* 2.7.x

依赖的 Python 模块：

* lxml
* requests

入口脚本：

* fetch.py
